Calling pdflatex / bibtex


To translate a LaTeX file: 
pdflatex mypaper.tex
bibtex mypaper.aux        (if using .bib file)
pdflatex mypaper.tex
In the tex file: 
\bibliography{mypaper}
In the bib file use keywords "and", "and others" or {...} inside "..." like this: 
 
 author= {Samuel L. Smith and Quoc V. Le},
 author= "Pokholok and others",
 author  "{Pokholok et al.}",

To translate a file using psTricks: 
pdflatex -shell-escape source.tex
The file should contain 
\usepackage[pdf]{pstricks}%
\usepackage{pst-node,pstricks-add,pst-plot}%
First line should be %&pdflatex (or nothing, but not %&latex, this will give dvi output)
