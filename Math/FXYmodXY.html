F[X,Y] / (XY) over a field F

Polynomials in two variables X, Y modulo the ideal generated by XY.

Reduced

Definition
Yes.

Integral Domain

Definition
No: X * Y = 0.

Unique Factorization Domain

Definition
No. (not integral)

Dedekind

Definition
No. (not integral)

Principal Ideal Domain

Definition
No. (not integral)

Field

Definition
No. (not integral)

Characteristic

Definition
char(F).

Cardinality

Definition
Infinite. (all powers of X are different)

Local Ring

Definition
No: (X, Y) and (X-1, Y) are different maximal ideals.

Discrete Valuation Ring

Definition
No. (not local)

Regular Local Ring

Definition
No. (not local)

Noetherian

Definition
Yes: If R is a Noetherian ring and I is an ideal, then the factor ring R/I is also Noetherian.

Artinian

Definition
No. (The Wikipedia page claims: A finitely generated F-algebra R is Artinian if and only if R is finitely generated as F vector space.)
