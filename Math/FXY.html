F[X,Y] (Polynomials in two variables over a field)

Reduced

Definition
Yes. (even integral)

Integral Domain

Definition
Yes.

Unique Factorization Domain

Definition
Yes: F is UFD, and for a UFD R also R[X] is UFD.

Dedekind

Definition
No: Dedekind + UFD would be equivalent to PID, but this is not PID.
Example of ideal that is not product of prime ideals: (X^2, Y^2).

Principal Ideal Domain

Definition
No: (X,Y) is not a principal ideal.

Field

Definition
No.

Characteristic

Definition
char(F).

Cardinality

Definition
Infinite.

Local Ring

Definition
No: (X, Y) and (X-1, Y) are different maximal ideals.

Discrete Valuation Ring

Definition
No. (not local)

Regular Local Ring

Definition
No. (not local)

Noetherian

Definition
Yes: R=F[X] is a Noetherian ring, hence R[Y] = F[X,Y] is Noetherian by the Hilbert basis theorem.

Artinian

Definition
No. (The Wikipedia page claims: A finitely generated F-algebra R is Artinian if and only if R is finitely generated as F vector space.)
